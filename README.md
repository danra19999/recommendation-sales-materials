# recommendation-sales-materials　
ドシステム開発サービス 営業資料

国内最大級プラットフォームで月間2,000万人が使うシステムを開発した
エンジニアによる、本格的なレコメンド機能開発サービスの営業資料です。

## 📁 ファイル構成

- **presentation.html** - 営業用プレゼン資料（18スライド）
  - EC市場とレコメンドの重要性
  - 競合との比較（パッケージ型、大手SIer）
  - 開発実績と技術力
  - 導入効果とROI
  - サービス内容と料金

- **roi-calculator.html** - ROI計算ツール
  - 顧客の数値を入力して投資回収期間を算出
  - 複数シナリオでの効果シミュレーション
  - PDF出力、クリップボードコピー機能

## **Privateリポジトリで知り合いに共有する方法**

1. リポジトリをPrivateで作成
2. Settings → Collaborators
3. 「Add people」で知り合いのGitHubアカウントを追加
4. 知り合いに通知が届き、アクセス可能に

または、知り合いがGitHubアカウントを持っていない場合：
- GitHub Pagesで公開（URLを知っている人だけアクセス可能）
- またはHTMLファイルを直接ダウンロードしてもらう
